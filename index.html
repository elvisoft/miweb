Clon de Uber

Clon de Uber completo, aplicacion para celulares Android y Iphone hecho en React Native para el front y Nestjs para el backend. Base de datos MySql. La aplicacion tiene roles de: Conductor, Chofer y Administrador. Tambien tiene el modulo de Calificaciones del chofer y el cliente, tambien tiene historial de viajes, trazado de rutas origen, destino, se muestra en el mapa los choferes disponibles. Se puede registrar con email y password (con Google, Facebook se agregara muy pronto). Se puede solicitar un viaje, y se notifica al chofer mas cercano, si acepta el viaje, se notifica al cliente. Se puede cancelar el viaje, y se notifica al chofer. Se puede finalizar el viaje, y se notifica al cliente. Se puede ver el historial de viajes, y se puede calificar al chofer. Al finalizar el viaje se puede pagar en: Efectivo, Mercado Pago
